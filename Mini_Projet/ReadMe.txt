Exercice 1 du Mini_Projet

Que fait cette page?

Elle permet de noté les différents campus de l'Epsi avec le système de notation de l'Eurovision

Comment faire fonctionner le miniprojet:

-Ouvrez le node.js command prompt
-dirigez vous dans le dossier de l'exercice (celui ou vous vous situez en théorie)
-dans le command prompt de node: entrez la commande suivante: npm init, appuyez sur enter jusqu'o "is this ok?" dans le terminal, réappuyez sur entrez.
-toujours dans le command prompt: entrez la commande suivante: npm install express
-toujours dans le command prompt: entrez la commande suivante: npm install fs
-Toujours dans le command prompt: entrez la commande suivante: node serveur.js
-Le commande prompt devrais indiqué qu'il écoute sur le port 1025
-Allez dans votre navigateur internet préféré et tappez cette Url: http://localhost:1025/?oui=note
-séléctionnez un des bouton
-Entrez les notes pour toutes les écoles, confirmez avec le Envoyez
-faite de même avec toutes les autres écoles
-si vous voulez voir les notes cliquez sur voir notes (ATTENTION: voir notes ne confirme pas l'envois de notes entrez précédement, seul envoyez confirme les notes)
