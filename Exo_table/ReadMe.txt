Exercice: tables dansantes

Que fait la page?:

Une page affiche une table dans laquel on peu definir combien de points et de tables
nous voulons et relies les points entre eux

comment le faire fonctionner:

Lancer le fichier index, ici vous pouvez renseigner le nombre de point et le nombre de 
tables que vous voulez. appuyez ensuite sur le bouton.
