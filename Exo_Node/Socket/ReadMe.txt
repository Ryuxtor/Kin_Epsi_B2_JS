Exo Socket.io

Que fait cette page?

Cette page simule une chatbox

comment faire fonctionner la page?

-Ouvrez le node.js command prompt
-dirigez vous dans le dossier de l'exercice (celui ou vous vous situez en théorie)
-toujours dans le command prompt: entrez la commande suivante: npm install express
-toujours dans le command prompt: entrez la commande suivante: npm install socket.io
-Toujours dans le command prompt: entrez la commande suivante: node index.js
-Le commande prompt devrais indiqué qu'il écoute sur le port 3000
-Allez dans votre navigateur internet préféré et tappez cette Url: http://localhost:3000/?
-ici vous devriez voir le chat et vous pouvez envoyez des messages
